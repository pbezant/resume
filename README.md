# Preston Bezant - Resume & Portfolio

This project is built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Project Structure

- `src/pages/`: Contains the route definitions.
  - `index.astro`: The main resume page.
  - `case-studies/`: Detailed case study pages.
- `src/components/`: Reusable UI components (Header, Footer, etc.).
- `src/layouts/`: Global layout templates.

## Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Start the development server:**

   ```bash
   npm run dev
   ```

3. **Build for production:**

   ```bash
   npm run build
   ```

4. **Preview the production build:**

   ```bash
   npm run preview
   ```
