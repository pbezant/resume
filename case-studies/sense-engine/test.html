The Triangulation Challenge

Warehouses are hostile environments for RF signals. Metal racks cause multipath interference, making raw RSSI data unreliable.

We couldn't just average the signals. The Sense Engine implements a Weighted Centroid Localization (WCL) algorithm with dynamic filtering. It ingests streams from Cisco, Moko, and Custom gateways, normalizing their different signal characteristics before calculating position.

function calculatePosition(beacons) {
  // 1. Filter weak or stale RSSI signals
  const valid = beacons.filter(b => 
    b.rssi > -90 && 
    (Date.now() - b.lastSeen) < 5000
  );

  // 2. Weighted Centroid Logic
  let xNum = 0, yNum = 0, weightSum = 0;
  
  valid.forEach(b => {
    const weight = Math.pow(10, b.rssi / 20);
    xNum += b.x * weight;
    yNum += b.y * weight;
    weightSum += weight;
  });

  // 3. Return normalized (x,y)
  return {
    x: xNum / weightSum,
    y: yNum / weightSum
  };
}
